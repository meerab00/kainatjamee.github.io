# My Portfolio Website

A personal portfolio website hosted on GitHub Pages.

## 🚀 Live Site

Visit: `https://yourusername.github.io`

## 📁 File Structure

```
portfolio/
├── index.html     ← Main page
├── style.css      ← All styles
├── script.js      ← Animations & interactions
├── resume.pdf     ← Your resume (add this!)
└── README.md      ← This file
```

## ✏️ Customization Checklist

Open `index.html` and update:

- [ ] `<title>` — your name
- [ ] `.nav-logo` — your initials
- [ ] Hero section — your name & description
- [ ] About section — your bio and stats
- [ ] Skills section — your actual tech stack
- [ ] Projects section — your real projects with links
- [ ] Contact section — your email & social links
- [ ] Footer — your name

Replace the photo placeholder with a real `<img>` tag.

Add your `resume.pdf` to the folder root.

## 🌐 Deploying to GitHub Pages

1. Create a repo named `yourusername.github.io`
2. Upload all files to the repo
3. Go to **Settings → Pages**
4. Set source to **main branch / root**
5. Your site is live in ~1 minute!

## 🎨 Color Customization

Edit the CSS variables in `style.css`:

```css
:root {
  --bg: #0a0a0a;         /* background */
  --fg: #f0ede6;         /* text */
  --accent: #e8c547;     /* yellow accent */
  --accent2: #ff6b35;    /* orange accent */
}
```
